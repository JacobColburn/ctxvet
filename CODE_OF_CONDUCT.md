# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the work. Harassment, personal attacks, and demeaning behavior aren't welcome here.

Report unacceptable behavior by opening a confidential contact via the repository owner. Maintainers may remove comments, commits, issues, and contributions that violate this code, and may ban repeat or severe offenders.
